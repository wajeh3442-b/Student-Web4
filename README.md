📄 README - Squares of Numbers (HTML Project)
📌 Overview

This project is a simple HTML webpage that displays the squares of numbers from 1 to 20. It demonstrates the use of superscript formatting in HTML to represent mathematical expressions clearly.

🛠 Features
Displays numbers from 1 to 20
Uses <sup> tag for superscript formatting
Clean and simple layout
Easy-to-read mathematical presentation
Beginner-friendly HTML structure
💻 Technologies Used
HTML5
📂 File Structure
index.html – Main file containing the code
🚀 How to Run
Save the file as index.html
Open it in any web browser
View the squares of numbers from 1 to 20
🎯 Purpose

This assignment helps in understanding:

How to use superscript (<sup>) in HTML
Basic formatting of mathematical expressions
Structuring content in a readable way
Beginner-level webpage creation
📚 Learning Outcome

After completing this project, you will understand how HTML can be used to present simple mathematical data in a clean and structured format.
